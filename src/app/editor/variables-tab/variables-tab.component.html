Custom Variables

Define a placeholder here, then type its token (e.g. {{ '{{bookingId}}' }}) into any text field in the editor. The default value fills the Preview tab and pre-fills the Send dialog, where it can be overridden per send. Variables belong to this email — they are stored when you save it with the Save button in the top bar.
